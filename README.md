# boite-m-dicament-intelligent
Notre projet consiste à développer une boîte à médicaments intelligente permettant d’aider les patients à prendre leurs médicaments à l’heure grâce à un système automatique de rappel.
À l’heure programmée, la carte Arduino active le buzzer et l’écran OLED affiche les informations nécessaires,le capteur ultrason détecte la main de l’utilisateur pour confirmer la prise 
du médicament et arrêter l’alarme, tandis que le capteur DHT22 surveille la température et l’humidité.
Enfin, le module LoRa assure la communication sans fil et la transmission des données à longue distance.
